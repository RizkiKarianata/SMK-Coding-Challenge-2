# Challenge 2
Challenge 2 merupakan tugas kedua dari pelatihan SMK Coding Online berbasis Mobile yang menggunakan Android Kotlin.

# Usage
* Install Java SE Development Kit
* Install Android Studio
* Configure Android Studio
* Open Android Studio
